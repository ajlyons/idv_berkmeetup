HTML Widgets: Leaflet Demo

leaflet is a JavaScript library for creating dynamic maps that support panning and zooming along with various annotations like markers, polygons, and popups. Viewing map tiles requires an internet connection.

library(leaflet)
orstationc <- read.csv("orstationc.csv")
pal <- colorQuantile("YlOrRd", NULL, n = 8)
leaflet(orstationc) %>% addTiles() %>% addCircleMarkers(color = ~pal(tann))

Support for other map tiles, markers, popups, sp objects (points, lines, polygons), and GeoJSON

More info: http://rstudio.github.io/leaflet/
